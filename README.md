Данный проект демонстрирует навыки работы с системой контроля версий Git, языком разметки Markdown, командной строкой и организацией структуры проекта. Включает в себя frontend-часть (HTML-страницы), backend-часть (консольное приложение на C#) и практические упражнения с терминалом.
