## Списки

### Маркированный список
- Элемент 1
- Элемент 2
- Элемент 3

### Нумерованный список
1. Первый пункт
2. Второй пункт
3. Третий пункт

### Вложенный список
- Основной пункт 1
  - Вложенный пункт 1.1
  - Вложенный пункт 1.2
- Основной пункт 2
  1. Вложенный нумерованный 2.1
  2. Вложенный нумерованный 2.2
